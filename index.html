Event Details

Event Name: ZeroBugs Debugging Event
Date: 26th April 2025
Organized By: CSE 3rd years
Technologies Involved: Python, C, HTML

ZeroBugs is a hands-on event where participants will compete to fix coding bugs across multiple languages like Python, C, and HTML. This event aims to enhance debugging skills and problem-solving techniques.

Debugging Questions

  • Question 1: Python

    Debug the following Python code:

    def add_numbers(a, b):
    result = a + b
    print("The sum is: " result)
    
add_numbers(5, 7)

    Hint: Check for syntax errors and how the result is printed.

  • Question 2: C

    Debug the following C code:

    #include 
int main() {
    int x = 10;
    int y = 20;
    printf("The sum of %d and %d is %d", x, y);
    return 0;
}

    Hint: Ensure correct usage of format specifiers.

  • Question 3: HTML

    Debug the following HTML code:

    
    
        My Webpage
    
    
        
    Welcome to My Webpage
    
        
    This is a paragraph without a closing tag.

    Hint: Check for missing closing tags and proper nesting of elements.
